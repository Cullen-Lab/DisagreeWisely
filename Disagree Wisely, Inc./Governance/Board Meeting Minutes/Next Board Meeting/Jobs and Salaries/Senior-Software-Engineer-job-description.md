# Senior Software Engineer
## Disagree Wisely, Inc.

**Position Title:** Senior Software Engineer  
**Reports To:** Chief Technology & Operations Officer (CTOO)  
**FLSA Status:** Full-time, Exempt (≈ 40 hrs/week)

## Position Summary

As Senior Software Engineer at Disagree Wisely, you lead the scaling and expansion of the established Sway platform—a sophisticated AI-enhanced educational collaboration system already serving thousands of students. You manage a team of 3+ software engineers while working directly with the CEO and CTOO to enhance existing enterprise-grade infrastructure, optimize performance at scale, and build new advanced features for platform growth.

## Key Responsibilities

### Technical Leadership & Scaling
- Scale existing Flask 3.0+ microservices architecture to support hundreds of thousands and eventually millions of concurrent users
- Optimize and enhance existing student matching algorithms for improved performance and new matching criteria
- Expand multi-LLM integration capabilities and implement advanced AI monitoring and analytics systems

### Platform Enhancement & Infrastructure
- Enhance existing WebSocket systems and Redis architecture for improved scalability and new collaboration features
- Optimize Google Cloud Platform infrastructure and implement advanced auto-scaling and monitoring solutions
- Expand background processing capabilities and implement new distributed task workflows for platform growth

### Team Management & Strategic Collaboration
- Manage and mentor team of 3+ software engineers across full-stack development
- Collaborate directly with CEO and CTOO on technical strategy and platform roadmap
- Lead technical hiring, code reviews, and engineering best practices implementation

### New Feature Development & Research Integration
- Design and implement new educational features and advanced analytics capabilities
- Build comprehensive data pipeline systems for research insights and platform optimization
- Create new integration frameworks for expanded research partnerships and institutional tools

## Required Qualifications

- **7+ years of professional software development experience** with senior-level expertise
- **Expert-level Python/Flask and JavaScript/TypeScript** with microservices architecture experience
- **Google Cloud Platform expertise** including Firebase, Kubernetes, and containerization
- **Real-time systems development** including WebSockets, message queues, and event-driven architecture
- **Team leadership experience** managing 3+ engineers with mentoring responsibilities
- **Multi-LLM integration experience** with AI monitoring and performance optimization
- **Educational technology or collaborative platform development** background preferred

## Work Environment

Fully remote; no routine travel required

## Classification & Compensation

- Exempt salaried position; at-will employment
- Salary set by the Board using IRC §4958 comparability data (Senior Software Engineer, Technical Lead) and reviewed annually
